

# Running the tutorials

If you want to run the tutorials on your local computer you will need to install the required python packages. We suggest to follow the steps below

* Download and install with conda the igwn-py311 environment from [this page](https://computing.docs.ligo.org/conda/environments/igwn-py311/).

* Activate the conda environment with 

```
conda activate igwn-py311
```

* Clone this repository with 

```
git clone https://github.com/apalmese/GW_teaching_material.git
```

* You are ready to go!
